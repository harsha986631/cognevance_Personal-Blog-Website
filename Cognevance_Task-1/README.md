# 📝 Personal Blog Website (Flask)

A simple and responsive Personal Blog web application built using
**Flask**, **SQLite**, **SQLAlchemy**, and **Flask-Login**.

## 🚀 Features

-   User Registration & Login
-   Secure Password Hashing
-   Create, Read, Update and Delete (CRUD) blog posts
-   SQLite database with SQLAlchemy
-   Responsive UI
-   Flash messages
-   Custom 403 & 404 error pages

## 🛠 Technologies Used

-   Python 3
-   Flask
-   Flask-SQLAlchemy
-   Flask-Login
-   SQLite
-   HTML5
-   CSS3
-   JavaScript

## ⚙️ Installation

``` bash
pip install flask flask_sqlalchemy flask_login werkzeug
python app.py
```

Open: `http://127.0.0.1:5000`

## 🔒 Security

-   Password hashing
-   Login required for CRUD operations
-   Users can manage only their own posts
