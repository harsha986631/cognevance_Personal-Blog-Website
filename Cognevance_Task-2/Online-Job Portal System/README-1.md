# Job Portal System

**Author:** N. Harsha Manikanta

## Overview

The **Job Portal System** is a Flask and SQLite based web application
where employers post jobs and job seekers search and apply.

## Features

-   User Registration & Login
-   Role-Based Access (Employer / Job Seeker)
-   Job Posting & Management
-   Job Search & Filtering
-   Job Applications
-   Profile Management
-   REST API

## Technologies

-   Python
-   Flask
-   SQLite
-   HTML5
-   CSS3
-   Bootstrap 5
-   Jinja2

## Installation

``` bash
pip install flask
python app.py
```

Open http://127.0.0.1:5000
